# loosilo

**Developer tooling for humans and AI agents**

</div>

---

We build focused CLI tools that remove friction from daily development workflows.
No bloat, no magic, just tools that do one thing well and compose naturally.

## Tools

| Tool | Description |
|------|-------------|
| [**contextswitch**](https://github.com/loosilo/contextswitch) | Switch your entire dev environment (AWS, K8s, venv, git, env vars) in one command |
| [**contextgraph**](https://github.com/loosilo/contextgraph) | Context graph tooling |

## Philosophy

- **One command** over multi-step rituals
- **Shell-native** — integrates with your existing workflow, doesn't replace it
- **Safe by default** — drift detection, dangerous-context guards, no surprises
- **Composable** — YAML config, plugin system, scriptable
